Javascript fundamentals

Array methods

array.push() --- adds to end, returns the new length of the array

array.pop() --- remove from end, returns the removed element

array.unshift() --- add to start, returns the new length of the array

array.shift() --- remove from start, returns the removed element

array1.concat(array2) --- merge two arrays

array.includes('cat') --- returns boolean

array.includes('cat', 3) --- check if the array has 'cat' after index of 3 (inclusive)

array.indexOf('cat') --- returns the first index at which a given element is found in the array, or returns -1 if it's not present

array.indexOf('cat', 3) --- returns the first index after the given index

array.reverse() --- reverses an array in place. Mutates the original array. Can combine with join().

array.join() --- creates and returns a new string by joining all elements in an array into one string, separated by commas or a specified separator string
